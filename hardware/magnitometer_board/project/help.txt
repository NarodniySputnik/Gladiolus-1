Справочник по командам
Общие команды:
Команда	Описание	Формат
help	Вывести список доступных команд	help
status	Вывести текущие настройки датчиков и часов	status

Управление магнитометром (QMC5883L):
Команда	Описание	Пример
mag read	Считать и вывести данные с магнитометра (X, Y, Z)	mag read
mag temp	Считать и вывести температуру магнитометра	mag temp
mag set odr <value>	Установить частоту опроса магнитометра (ODR). Поддерживаются: 10, 50, 100, 200	mag set odr 50
mag set range <value>	Установить диапазон измерений магнитометра (2 или 8 Гаусс)	mag set range 2
mag set mode <value>	Установить режим работы магнитометра: 0 – standby, 1 – continuous	mag set mode 1
mag get config	Вывести текущие настройки магнитометра	mag get config

Настройка будильника (PCF8563):
Команда	Описание	Пример
rtc get time	Получить текущее время в формате Unix	rtc get time
rtc set alarm <sec>	Установить интервал будильника в секундах	rtc set alarm 60
rtc get alarm	Получить текущий интервал будильника	rtc get alarm

Дополнительные команды:
Команда	Описание	Пример
reset	Перезагрузить устройство	reset
